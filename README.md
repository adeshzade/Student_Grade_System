# 🎓 Student Grade Tracker System

A console-based **Student Grade Tracker System** developed using **Core Java** to manage and monitor student academic records. The application allows users to add, view, search, update, and delete student records while automatically calculating grades based on marks. It also generates a summary report that provides useful insights such as average marks, highest scorer, lowest scorer, and the total number of students.

This project is designed to demonstrate the practical implementation of **Object-Oriented Programming (OOP)** concepts and Java Collections. It is suitable for beginners who want to build a real-world Java application while learning data management and program organization.

---

## 🚀 Features

* ➕ Add new student records
* 📋 Display all student records in a formatted table
* 🔍 Search students using their roll number
* ✏️ Update student name and marks
* ❌ Delete student records
* 🏆 Automatically assign grades based on marks
* 📊 Generate summary reports including:

  * Total number of students
  * Average marks
  * Highest marks
  * Lowest marks
* 🖥️ Simple menu-driven console interface

## 📖 How It Works

1. Launch the application.
2. Select an option from the menu.
3. Add student details including roll number, name, and marks.
4. The system automatically calculates the student's grade.
5. View, search, update, or delete student records as needed.
6. Generate a summary report to analyze overall student performance.
